# Support

For support with this project, please use the following channels:

- **Enterprise Support**: <https://owncloud.com/contact-us/>
- **Community discussions**: https://github.com/orgs/owncloud/discussions
- **Matrix Chat**: <https://app.element.io/#/room/#owncloud:matrix.org>
- **Documentation**: <https://doc.owncloud.com>

This project integrates with [ownCloud Infinite Scale (oCIS)](https://github.com/owncloud/ocis);
for support with oCIS itself, see its own repository and community channels.

Please do not use GitHub issues for general support questions.
